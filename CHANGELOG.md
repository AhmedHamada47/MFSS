# Changelog

## v4.2.0 (2026-06-21)
- Initial public release
- Support: Cloudflare R2, AWS S3, Azure Blob, Google Cloud, Local
- Database: SQL Server, PostgreSQL, MySQL, SQLite
- Features: Circuit breaker, retry, dead letter queue, SHA-256 checksums
- CLI: --dry-run, --fresh, --parallel, --mode rollback, --config
- Environment variable support for secrets
- Webhook notifications (Discord, Slack, Teams)
- CSV export, audit trail, log files
- Auto-generates appsettings.json on first run
